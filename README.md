MVP моего личного пет-проекта version 1.1 (в разработке)
- Статистика
- Kanban доска
- Календарь (день, неделя, месяц, год)

Фишки:
- Теги с цветовой связкой
- drag-and-drop + анимации для недельного календаря
- Горячие клавиши копирование событий при перетаскивании + Alt, отмена действия Ctrl+Z, переключение между неделями кваитурными стрелочками
- Редактирование задач и событий (настройка названия, описания, времени, даты, приоритета)
- Все в одном темном стиле, минимализм
- Выпадающая при наведении навигационная панель с переключением страниц
- Плавные анимации при переключении
- Статистика показывающаяя занятость в %, каждого дня и график времени по тегам
- 7-24 и 0-24 режим (из каледаря можно вырезать 00-00 - 07-00 промежуток) одной кнопкой, так как часто это время уходит на сон, решения принято для экономии места на доске, если у вас обычный график

Запуск проект:
docker-compose up --build
