# Personal Flow — MVP v1.2

> **Персональный органайзер** с Канбан-доской, календарем и аналитикой.  
> Проект создан для удобного планирования дня, недели и месяца с упором на визуальную чистоту и скорость взаимодействия.

---

## 🚀 Возможности (v1.2)

### 📅 Календарь
- Режимы просмотра: день, неделя, месяц, год
- Недельный календарь с **drag-and-drop** и анимациями
- Копирование событий при перетаскивании (`Alt` + Drag)
- Отмена действий (`Ctrl+Z`)
- Навигация по неделям с помощью клавиш-стрелок
- Кнопка переключения режима отображения: **7–24** (основное время) / **0–24** (полные сутки)
- Упрощённый выбор даты (горизонтальный ряд ближайших дней)
- Создание и редактирование событий с выбором времени через **двойной ползунок**

### 📋 Канбан-доска
- Управление задачами в стиле "To Do / In Progress / Done"
- Доски с вкладками (можно создавать несколько досок)
- **Inbox** — единый список для сбора всех задач
- Перетаскивание задач из Kanban Inbox прямо на событие в календаре
- Drag-and-drop категорий из Inbox на события календаря для быстрого создания связей

### 📊 Статистика
- Процент занятости по дням (сколько времени запланировано)
- График распределения времени по тегам
- Наглядная аналитика вашей активности за неделю/месяц

### 🏷️ Теги
- Цветовая маркировка событий и задач
- Кастомные иконки для тегов (выбор из библиотеки)
- Панель управления тегами с прокруткой

### 🎨 Персонализация
- Установка фонового изображения с регулировкой прозрачности
- Настройки фона через шестерёнку в левом верхнем углу

### ✏️ Редактирование событий
- Название и описание
- Выбор даты из ближайших дней
- Выбор времени через двойной ползунок (начало/конец)
- Выбор тега
- Повторение событий (еженедельно, с выбором дней недели и даты окончания)

### 🧭 Интерфейс
- Выпадающая навигационная панель (появляется при наведении)
- Полностью тёмная тема, минималистичный дизайн
- Плавные анимации при переключении между страницами

---

## 🛠️ Запуск проекта

```bash
docker-compose up --build
```

После запуска:
- Frontend: http://localhost:8080
- Backend API: http://localhost:8000

---

## ⌨️ Горячие клавиши

- `Alt` + Drag — копировать событие при перетаскивании
- `Ctrl+Z` — отменить действие
- `←` / `→` — навигация по неделям/месяцам
- `Delete` / `Backspace` — удалить выбранное событие
- `Escape` — снять выделение
